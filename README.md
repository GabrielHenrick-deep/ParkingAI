# Aplicativo de Reservas de Vagas de Estacionamento Assistido por IA

Este projeto é um aplicativo de reservas de vagas de estacionamento que utiliza inteligência artificial para otimizar a alocação de vagas e melhorar a experiência do usuário.

## Funcionalidades

- **Busca de Vagas**: Usuários podem buscar vagas disponíveis em tempo real.
- **Reservas Inteligentes**: O aplicativo utiliza algoritmos de IA para sugerir vagas com base na localização do usuário e nas preferências pessoais.
- **Gerenciamento de Reservas**: Usuários podem visualizar, modificar ou cancelar suas reservas.
- **Notificações**: Alertas sobre a proximidade do horário da reserva ou mudanças de disponibilidade.
- **Interface Amigável**: Design intuitivo que facilita a navegação.

## Tecnologias Utilizadas

- **Front-end**: [Cordova](https://cordova.apache.org/) para desenvolvimento multiplataforma.
- **Back-end**: [Node.js](https://nodejs.org/) com [Express](https://expressjs.com/) e [Golang](https://golang.org) com Gin para construir a API.
- **Banco de Dados**: [PostgreSQL](https://www.postgresql.org/) para armazenamento de dados.
- **Inteligência Artificial**: Algoritmos de Machine Learning para análise de dados de estacionamento e predição de disponibilidade.
